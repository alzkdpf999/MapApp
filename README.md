### 💡프로젝트 소개:bulb:
MapApp은 식당, 카페, 술집에 대한 정보만들 다루어 가게에 위치에 도착하면 알람을 보내주는 프로그램입니다.
> 지도에서 가고자하는 가게들을 찾고서 친구들과 얘기하면서 가다가 갈 가게를 놓쳐버리는 상황을 생각해서 만들었습니다.
